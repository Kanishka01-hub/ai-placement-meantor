# AI Placement Mentor

AI Placement Mentor is a placement preparation web application built using React, Node.js, Express.js, and MongoDB.

## Features

- Login and Signup
- Student Dashboard
- Resume Checklist
- Placement Roadmap
- Company Application Tracker
- Profile Builder
- Resume Upload

## Tech Stack

- React.js
- JavaScript
- HTML
- CSS
- Node.js
- Express.js
- MongoDB
- Git & GitHub
- Vercel
